## Project Description
This project is a Java-based console application designed to simulate a hotel reservation system.
It allows customers to register, search available rooms, and book reservations.
Administrators can manage rooms and view all reservations.

## Technologies Used
- Java (Core Java, OOP concepts)
- Collections Framework
- Exception Handling

## Features
- Customer registration with email validation
- Room management (paid and free rooms)
- Room availability search by date
- Reservation creation and storage
- Console-based menu navigation
- Clean separation of model, service, and UI layers

## Design Approach
- Object-Oriented Programming principles
- Interfaces for abstraction
- Inheritance for room types
- Singleton pattern for service classes

## Plagiarism Declaration
This project was implemented independently.
Standard Java design patterns and concepts are used, but the code structure,
logic, and implementation are original.

## How to Run
1. Compile all '.java' files
2. Run the 'Hotel Application'class
